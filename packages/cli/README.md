## `bearer deploy`

Deploy a scenario

```
USAGE
  $ bearer deploy

OPTIONS
  -h, --help          show CLI help
  -i, --intents-only  Deploy intents only
  -s, --views-only    Deploy views only
```

_See code: [src/commands/deploy.ts](https://github.com/Bearer/bearer/blob/v0.41.6/src/commands/deploy.ts)_

## `bearer generate [NAME]`

Generate scenario intents or components

```
USAGE
  $ bearer generate [NAME]

OPTIONS
  -h, --help              show CLI help
  --blank-component
  --collection-component
  --root-group
  --setup
```

_See code: [src/commands/generate.ts](https://github.com/Bearer/bearer/blob/v0.41.6/src/commands/generate.ts)_

## `bearer help [COMMAND]`

display help for bearer

```
USAGE
  $ bearer help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v2.0.5/src/commands/help.ts)_
